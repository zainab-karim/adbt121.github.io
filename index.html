Bootcamp 2020: Bubble Popper

My first programming project, concluding the 2 week Programming Bootcamp at City 2020. This Processing project demonstrates a comprehensive understanding of the use of loops, if statements, user interactivity and audio files. I created a bubble pop game in which the main objective is to tap as many bubbles as you can in under 30 seconds. Every time a bubble is pressed, another bubble will appear on the screen in a random location and size. The users score will increase by one every time they score a point. At the end of the coundown, the users score will be displayed in a game over screen.

Check out the code on Github: https://github.com/zainab-karim/adbt121/
